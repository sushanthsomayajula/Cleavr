# website/

The static site for Cleavr (GNRHR / TNBC subtype expression atlas).

- `index.html` — full write-up, open this first
- `explorer.html` — interactive subtype/expression/structure/survival explorer

Both are plain files — double-click to open in a browser, no server needed. They reference `../results/`, `../code/`, and `../docs/`, so keep this folder inside the project alongside those directories rather than moving it on its own.

See the [project README](../README.md) one level up for the full folder structure, methodology notes, and how this relates to existing public tools (cBioPortal, GEPIA2, UALCAN, GTEx, UCSC Xena).
