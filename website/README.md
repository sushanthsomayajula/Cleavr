# website/

The static site for Cleavr.

- `index.html` — home / project pitch, open this first
- `gnrhr.html` — Case Study 01: GNRHR expression across TNBC subtypes, full write-up
- `explorer.html` — interactive subtype/expression/structure/survival explorer for Case Study 01

All three are plain files — double-click to open in a browser, no server needed. They reference `../results/`, `../code/`, and `../docs/`, so keep this folder inside the project alongside those directories rather than moving it on its own.

See the [project README](../README.md) one level up for the full folder structure, methodology notes, roadmap, and how this relates to existing public tools (cBioPortal, GEPIA2, UALCAN, GTEx, UCSC Xena).
